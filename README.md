# 🫀 CardioVision AI
### Advanced Heart Disease Risk Prediction System

CardioVision AI is a Machine Learning based healthcare application designed to predict the risk of cardiovascular disease using patient health parameters. The system analyzes clinical metrics and lifestyle factors to estimate the probability of heart disease and provide health recommendations.

---

## 🚀 Project Overview

Heart disease is one of the leading causes of death globally. Early risk detection can significantly improve prevention and treatment outcomes.

CardioVision AI leverages **Machine Learning algorithms** to analyze patient health metrics such as age, blood pressure, cholesterol levels, heart rate, and lifestyle habits to estimate cardiovascular risk.

The system provides:

- AI based risk prediction
- Health dashboard visualization
- Lifestyle recommendations
- Interactive web interface

---

## 🧠 Machine Learning Model

The project uses a trained ML model to predict cardiovascular risk.

Algorithms used during experimentation:

- Random Forest Classifier
- Logistic Regression
- XGBoost

The final model is trained on the **UCI Heart Disease Dataset**.
